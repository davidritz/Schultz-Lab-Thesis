## "Common regulatory mutation increases single-cell survival to antibiotic exposures in Pseudomonas aeruginosa" by Ritz et al  
### The publication's analysis code for figure generation can be found here: https://github.com/schultz-lab/Response-Dynamics.  
- The "RPI" directory is Raspberry Pi code to run a custom-built chemostat for bacterial competitions.  
- The "Mother Machine" directory is used for single-mother bacterial cell tracking in mother machine microfluidic devices (sample analysis video below tracks length and mexXY expression of P. aeruginosa cells).  
- The "measuring resistance" directory quantifies antibiotic resistance of populations of bacteria.
